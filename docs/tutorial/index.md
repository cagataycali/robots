# Tutorial

Welcome to the Strands Robots tutorial. By the end, you'll go from zero to building AI-powered robot agents.

## What You'll Build

Over 9 chapters, you'll:

1. Create your first simulated robot
2. Understand physics simulation
3. Connect AI policies that tell the robot *how* to move
4. Use natural language to control robots through AI agents
5. Coordinate multiple robots
6. Record demonstrations for imitation learning
7. Train your own policies
8. Deploy to real hardware
9. Use advanced features like DreamGen and GR00T

## Requirements

```bash
pip install "strands-robots[all]"
```

That installs everything — simulation, training, all policy providers.
