# Installation

!!! info "Coming Soon"
    This page is under active development.

## Requirements

- Python 3.10+
- pip or conda

## Install from PyPI

```bash
pip install strands-robots
```

## Install from Source

```bash
git clone https://github.com/strands-labs/robots
cd robots
pip install -e .
```

## Dependencies

- `strands-agents` — Agent framework
- `lerobot` — Hardware abstraction
- `torch` — Neural network runtime
- `opencv-python-headless` — Vision processing
