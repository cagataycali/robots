# Robot Factory

!!! info "Coming Soon"
    This page is under active development.

The Robot Factory provides pre-configured robot setups for common hardware.

## Supported Configurations

| Robot | Config | Description |
|-------|--------|-------------|
| SO-100 | `so100` | Single arm desktop robot |
| SO-101 | `so101_follower` | Single arm with follower |
| GR-1 | `fourier_gr1_arms_only` | Humanoid bimanual arms |
| Unitree G1 | `unitree_g1` | Humanoid robot platform |
